Data Types

What Are Data Types?

In computer science and computer programming, a data type - or simply “type” - is a classification of data which tells the compiler or interpreter how the programmer intends to use the data.

Examples of some data types are:

  • Booleans (true, false)
  • String (text defined within quotes "")
  • Numbers (1, 3.14, 0.25)


Some programming languages (not JavaScript) require assigning the scope of numerical data to define how that data will get stored:

  • Integers are only stored as whole values (0, 4, -3)
  • Floats are stored values that can have decimal places (0.25, 3.1415926, -95.6)

Integers use less memory than Floats.

When Do We Use Data Types?

When we are writing our applications, it is important to think about what we will be doing with our variables and data. The data type determines what type of values the variable can contain and what operations can be performed on it.

For example, in some languages, you can do mathematical calculations on numbers, but not on strings. A string can contain alphanumeric data meaning that there can be many different characters (a, 1, !, etc.), but they are all considered as if they were text, even if the characters are numbers.

Some programming languages require that you define the data type along with the name when you create the variable and before you do anything with it. But other languages, like JavaScript, are not so strict.

Control Structures

Control structures are how we introduce logical operations to our code based on the values of data types: sometimes when we only want something to happen if a certain condition is met, while other times we want to do an operation repeatedly. These two types of control - conditionals and loops - are foundational to programming.

Conditionals

Use a conditional, or if statement, to execute code based on some condition:

  • if (condition) {
  • // code here executes if the condition is true
  • }


if statements can be expanded using else statements and else if statements with other conditions for extra checks:

  • if (condition) {
  • // code here executes if the condition is true
  • } else if (otherCondition) {
  • // code here executes if the first condition is false and this second
  • // condition is true
  • } else {
  • // code here executes if the first condition and the second condition are
  • // false
  • }


Conditions

Conditions can be variables that are true or false, or can be statements that use comparison and logical operators. The operators compare two values to determine equality, inequality, or relative scale; some operators can be used with various data types (=== for equality), while others don’t make sense with some data types (is "red" > "blue"?).

Some of the most common comparison operators are:

Operator Description

  • a == b true if a is equal in value to b
  • a === b true if a is equal in type and value to b
  • a != b true if a not equal to value of b
  • a !== b true if a not equal to value and type of b
  • a > b true if a is larger than b
  • a >= b true if a is larger than or equal to b
  • a < b true if a is less than b
  • a <= b true is a is less than or equal to b
  • c && d true if both statement c and d are true
  • c || d true if either statement c or d (or both) are true
  • !c true if statement c is false

These operators can be combined to create complex statements.

Loops

Loops are used to repeatedly execute code, and one of the most powerful types of loop is a for loop. For example:

  • for (let i = 0; i < 3; i++) {
  • // the code here will be executed 3 times
  • }

for loops have three parts:

  • 1) START: Initialize a variable to some value, typically some kind of counter.

  • let i = 0;

  • 2)END: Keep executing the code inside the curly braces {} while this condition is still true.

  • i < 3;

  • 3)STEP: How to update to initialized variable after each time the code inside the curly braces {} is executed.

  • i++ which is shorthand for i = i + 1.
