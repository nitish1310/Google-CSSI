Scripting Language Overview

What is Scripting?

Scripting is something that happens behind the scenes of every interactive website. A script can:

  • determine how the website behaves in response to clicks or interactions made by the user.
  • automate the execution of tasks or events that happen on a site.
  • be opened and edited with the help of a basic text editor.


When Do We Use Scripting?

Scripts are not visible to the user, but they are the code that is written that makes a website responsive to the user’s actions. When a user clicks a button, hovers over an icon, or interacts in any way, a script can determine how the website behaves in response to the user’s action.

We use scripting in both the front- and back-end of our applications.

Scripting in Action

Scripting is divided into two parts: client-side (front-end) scripting and server-side (back-end) scripting.

Client-Side (Front-End) Scripting

Client-side (front-end) scripting is done in the client’s browser. The scripting language most popular for front-end is definitely JavaScript.

Be aware, the client-side (front-end) script code is not private and can be viewed by everyone who visits the site by looking in the developer tools of the browser.


Server-Side (Back-End) Scripting

Server-side (back-end) scripting is done on a server. This means that it’s not done on the user’s computer, but rather somewhere remote.

An example of this would be someone who is on their bank’s website. When the user is logging in to see their account information, a script on the back-end will get their bank account information and the user will see only the result of this script in their browser as a display of their accounts and information.

Server-side (back-end) scripts are more secure and these scripts are usually the ones that connect to and retrieve data from databases. Some popular server-side (back-end) scripting languages are:

  • PHP
  • Ruby
  • Python
  • JavaScript (this can be used for both front- and back-end)

Types of Programming Languages

There are two programming languages you will use in this course: JavaScript and Python. Programming languages are scripts that will run line by line until the program finishes or an error is encountered.

A different type of programming language called a “compiled” language must first completely convert the words used to write the scripts (a high-level language) to a different language understood by a computer (a low-level language) before the program can run. Examples of compiled languages are COBOL, C++, C#, Objective-C, Fortran, Go, Rust, and Swift.

A third type of language uses annotations to cause content to be rendered in a particular way. This is called “markdown” and is commonly used for documentation and notes. In fact, the content for this course has been completely written in markdown.
