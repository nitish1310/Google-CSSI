What Are Data Structures?

Data structures provide a means to manage large amounts of data efficiently. Data structures represent a collection of related data. Different kinds of data structures are suited to different applications and situations. Two types of structures are:

  • Arrays
  • Objects

When Do We Use Data Structures?

For anything more complicated than a single word or number, a data structure should be used. Note that data structures can store numbers, words, booleans, or a combination.

An array can be used for keeping track of a list of data (numbers, strings, booleans, other arrays, or a mix). An array holds the elements in a specific order and the data can be accessed by referencing the index of the storage spot (starting at 0).

  1. let seatingOrder = ["Bruce", "Peter", "Natasha", "Tony", "Steve", "Carol",
  2. "Vincent", "Nick"];

  3. // To access the name in the fourth position, reference the storage index
  4. seatingOrder[3];

If the data structure is more complicated or represents something, for example, a car, an object would be a good data structure for keeping track of the number of doors, color, transmission, etc. With an object, values can always be efficiently inserted and removed without relocating other values. The values can be referenced by referencing the data structure’s name and the associated key.

  1. let myCar = {
  2. wheels: 4,
  3. doors: 5,
  4. color: "red",
  5. transmission: "automatic",
  6. needAnOilChange: false
  7. };

  8. // To access the color of the car, reference the key "color"
  9. myCar["color"];
